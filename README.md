# space-invaders
Space Invaders in Go using an [Donburi ECS](https://github.com/yohamta/donburi/) and [Ebiten Graphics library](https://github.com/hajimehoshi/ebiten)

## Instructions
* Arrow keys to move left and right
* Space bar to fire
* Q to quit
* R to restart
